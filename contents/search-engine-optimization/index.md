# 트래픽을 쓸어 담는 검색엔진 최적화

## SEO 체크리스트

### 도메인 (domain)
보유 도메인 현황 및 운영에 관한 요소를 점검한다.

- [ ] 대표 도메인이 설정되어 있는가?
- [ ] 향후 도메인 운영 전략이 있는가?
- [ ] 서브 폴더 방식을 사용하는가?


### 접근성 및 색인 (accessibility & indexation)
웹 페이지가 검색봇에 발견되고 색인되기 위해 갖춰야 할 요소를 점검한다.

- [ ] robots.txt 파일을 갖고 있는가?
- [ ] robots.txt 파일에 의해 의도치 않게 검색봇의 접근이 막힌 페이지가 있는가?
- [ ] sitemap.xml 파일을 갖고 있는가?
- [ ] sitemap.xml 주소가 robots.txt 파일에 리스트되었는가?
- [ ] 404 에러 페이지가 있는가?
- [ ] 모바일 친화적인 반응형 웹 페이지인가?
- [ ] [AMP(Accelerated Mobile Pages)](https://amp.dev/)를 사용하는가?
- [ ] 크롤링 예산은 적정한가?


### 메타 태그: 타이틀 (meta tag: title)
구글의 검색 결과 페이지에 노출되는 타이틀 태그와 관련해서 점검한다.

- [ ] 주요 페이지에 타이틀 태그가 설정되었는가?
- [ ] 타이틀 태그가 중복되어 사용되는가?
- [ ] 글자 제한 수에 못 미치는 타이틀 태그가 있는가?
- [ ] 타이틀 태그가 페이지 내용을 명확히 소개하는가?
- [ ] 글자 제한 수를 넘어서는 타이틀 태그가 있는가?


### 메타 태그: 디스크립션 (meta tag: description)
구글의 검색 결과 페이지에서 노출되는 메타 디스크립션 태그와 관련해서 점검한다.

- [ ] 주요 페이지에 메타 디스크립션 태그가 설정되었는가?
- [ ] 메타 디스크립션 태그가 중복되어 사용되는가?
- [ ] 글자 제한 수에 못 미치는 메타 디스크립션 태그가 있는가?
- [ ] 글자 제한 수를 넘는 메타 디스크립션 태그가 있는가?
- [ ] 메타 디스크립션이 페이지 내용을 명확히 소개하며 클릭 유도 문구가 기입되었는가?


### 메타 태그: 캐노니컬 태그 (meta tag: canonical)
중복 콘텐츠 생성을 방지하고 신뢰를 높이는 캐노니컬 태그와 관련해서 점검한다.

- [ ] (PC)캐노니컬 태그를 사용하는가?
- [ ] (MOBILE)캐노니컬 태그를 사용하는가?


### 최적화: 이미지 (optimization: image)
콘텐츠 내 이미지가 검색엔진에 잘 노출되는지 현황을 점검한다.

- [ ] 이미지 사이트맵이 있는가?
- [ ] 100KB를 초과하는 이미지가 있는가?
- [ ] 이미지 대체 텍스트가 설정되었는가?
- [ ] 사용된 이미지 대체 텍스트가 이미지의 내용을 정확히 설명하는가?
- [ ] 텍스트로 작성 가능한 내용을 이미지로 사용하지 않았는가?
- [ ] 파비콘(favicon) 이미지를 사용하는가?
- [ ] 이미지에 불필요한 링크가 있는가?


### 최적화: 비디오 (optimization: video)
콘텐츠 내 비디오가 검색엔진에서 잘 노출되는지 현황을 점검한다.

- [ ] 비디오 사이트맵이 있는가?


### 최적화: 소셜 미디어 (optimization: social media)
웹 사이트와 소셜 미디어 간의 링크 및 최적화 상황을 점검한다.

- [ ] 오픈 그래프 태그를 사용하는가?
- [ ] 트위터 카드 태그를 사용하는가?
- [ ] 소셜 미디어를 통해 웹 사이트 콘텐츠가 공유되는가?
- [ ] 소셜 미디어 계정이 웹 사이트로 링크를 주는가?


### 링크와 내비게이션 (link & navigation)
온페이지 링크를 작업하고 내비게이션 현황을 점검한다.

- [ ] 내부 링크가 충분히 있는가?
- [ ] 앵커 텍스트 최적화를 하였는가?
- [ ] 외부 링크가 충분히 있는가?
- [ ] 콘텐츠 사일로가 있는가(코너스톤 페이지가 있는가)?
- [ ] CTA가 명확한가?
- [ ] 비 관련 사이트로부터 유입 링크가 있는가?
- [ ] 고유 태그 주소가 있고, 사이트맵에도 포함되었는가?
- [ ] 고유 카테고리 주소가 있고, 사이트맵에도 포함되었는가?
- [ ] GNB 메뉴를 텍스트로 구성하였는가?
- [ ] 흩어져 있는 정보(breadcrumbs)가 존재하며 명확한가?
- [ ] 푸터(footer) 메뉴를 텍스트로 구성하였는가?
- [ ] 메인 페이지, 카테고리 페이지, 상세 페이지 사이의 수직적 링크와 상세 페이지 간의 수평적 링크 구조가 잘 설계되었는가?
- [ ] 중요 페이지로의 클립 뎁스(click depth)가 3클릭 이내인가?


### URL
웹 사이트 내의 URL 사용 현황을 점검한다.

- [ ] URL의 길이가 적당한가?
- [ ] URL에 소문자만 사용되었는가?
- [ ] URL이 충분히 서술적인가?
- [ ] 절대 주소를 사용하고 있는가?
- [ ] URL에 사용된 단어가 -로 구분되었는가?
- [ ] 아스키 문자로만 주소를 구성하였는가?


### 설정: 애널리틱스 (setting: analytics)
웹 로그 분석을 위한 트래킹 코드 설치, 설정, 활용과 관련해 점검한다.

- [ ] 애널리틱스 트래킹 코드가 모든 웹 페이지에 설치되었는가?
- [ ] 애널리틱스에서 사이트 검색(site search)을 트래킹 하는가?
- [ ] 구글 광고가 연동되었는가?
- [ ] 필터를 이용해 사내 IP주소를 제외시키는가?
- [ ] UTM코드를 통해 다양한 마케팅 캠페인을 트래킹하는가?
- [ ] 매크로 전환에 대한 목표 설정이 제대로 되었는가?
- [ ] 마이크로 전환에 대한 목표 설정이 제대로 되었는가?


### 설정: 서치 콘솔 (setting: search console)
웹 수집 현황 분석을 위한 툴(구글 서치 콘솔)의 설치, 설정, 활용과 관련해 점검한다.

- [ ] 구글 서치 콘솔이 연동되었는가?
- [ ] 타깃 국가가 설정되었는가?
- [ ] 접근 가능한 웹 주소가 모두 등록되었는가?


### 설정: 마이 비즈니스 (setting: my business) 
지역 검색 및 신뢰도 향상을 위한 툴(구글 마이 비즈니스)의 설치, 설정, 활용과 관련해 점검한다.

- [ ] 구글 마이 비즈니스가 연동되었는가?
- [ ] 브랜드 검색 시 지식패널(knowledge panel)이 보이는가?
- [ ] 지식 패널에 구글 포스트 내용이 포함되었는가?


### 그 외 (others)
기본적인 SEO 요소를 점검한다.

- [ ] 보안 프로토콜을 사용하는가?
- [ ] Hreflang 태그를 사용하는가?
- [ ] 플래시(flash)를 사용하지 않는가?
- [ ] 스키마 마크업(schema markup)을 사용하는가?
- [ ] 3초 이내에 로딩되는가?
- [ ] 기존 페이지로부터 리디렉션 시 301 redirect 설정을 사용하는가?
- [ ] 임시 리디렉션에만 302 리디렉션을 사용하는가?
- [ ] 404 에러 등 특수 상황에 대한 개별 페이지를 만들어 리디렉션하는가?
- [ ] 페이지 내에서의 사이트 검색 기능 및 관련 페이지 링크를 제공하는가?
- [ ] CSS, JS를 외부 파일로 사용하는가? (페이지의 HTML 내부에 사용하지 않는가?)
- [ ] JS의 경우 닫는 </body> 바로 위에 작성하였는가?
